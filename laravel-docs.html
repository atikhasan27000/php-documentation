Artisan Console

Creating a Laravel Project

composer create-project laravel/laravel example-app
php artisan serve
php artisan serve --host=0.0.0.0 --port=8080

PHP built-in server

php -S 127.0.0.1:8000 -t public

Laravel can display an overview of your application's configuration, drivers, and environment via the about Artisan command.

php artisan about

To quickly generate a new controller, you may run the make:controller Artisan command. By default, all of the controllers for your application are stored in the app/Http/Controllers directory:

php artisan make:controller UserController

Create Symlink: First, you need to link from public folder to storage/app/public folder. For this run the following command in terminal:

php artisan storage:link

This comment should be executed if anything in the .env file is updated.:

php artisan config:cache

generate Laravel new APP_KEY=

php artisan key:generate

Set Bangladesh timezone Asia/Dhaka (GMT+6)

APP_TIMEZONE=Asia/Dhaka

Create Laravel `model, migration, seed, controller` in 1 command.

php artisan make:model ModelName -mcs

Databases & Migrations

Create a new migration file.

php artisan make:migration create_users_table

Run all pending migrations to apply changes to the database.

php artisan migrate

Rollback the last database migration.

php artisan migrate:rollback

Rollback all database migrations.

php artisan migrate:reset

Rollback and re-run all migrations. This is useful for refreshing your database schema.

php artisan migrate:refresh

Show the status of each migration (whether they have been run or not).

php artisan migrate:status

Drop all tables and re-run all migrations from scratch.

php artisan migrate:fresh

Create the migration repository (the migrations table) if it doesn't exist.

php artisan migrate:install

Rollback a specific number of migrations.

php artisan migrate:rollback --step=<number>
php artisan migrate:rollback --step=2

Database Commands

Run the database seeders to populate your database with test data.

php artisan db:seed

Create a new seeder class.

php artisan make:seeder
php artisan make:seeder UserSeeder

Drop all tables, views, and types from the database, but do not run migrations.

php artisan db:wipe

Run the database migrations and seed the database with the data provided in seeders.

php artisan migrate --seed

Display the current database configuration.

php artisan db:show

Dump the current database schema and prune all migrations.

php artisan schema:dump

Monitor your database for connection issues (available in Laravel Vapor or Forge).

php artisan db:monitor

Application Configuration Cache

Make Cache

php artisan config:cache
php artisan route:cache
php artisan view:cache
php artisan event:cache
php artisan optimize

Clear Cache

php artisan optimize:clear
php artisan route:clear
php artisan config:clear
php artisan view:clear
php artisan event:clear

Clear the application cache.

php artisan cache:clear

Remove an item from the cache by its key.

php artisan cache:forget {key}
php artisan cache:forget user:123

Create a migration for the cache database table.

php artisan cache:table

Create a cache file for faster configuration loading.

php artisan config:cache

Remove the configuration cache file.

php artisan config:clear

Create a route cache file for faster route registration.

php artisan route:cache

Remove the route cache file.

php artisan route:clear

Compile all of the Blade templates into view cache.

php artisan view:cache

Clear all compiled view files.

php artisan view:clear

Clear a specific cache item:

php artisan cache:forget user:123

Optimize performance by caching config and routes:

php artisan config:cache
php artisan route:cache
